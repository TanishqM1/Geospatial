Nearest Responder Dispatch

Sidecar demo: random BC positions for five responders and an incident, snapped on the server to roads, then nearest unit by driving time via the Geospatial API.

Responder and incident positions

Five responders and one incident are placed in southern BC. Coordinates are randomized on land-ish bounds, then the server snaps them to the road network before routing. Set GEOSPATIAL_URL if the Geospatial API is not on http://127.0.0.1:8080.
